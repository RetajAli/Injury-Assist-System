# Source Code

This folder contains the complete Java source code for the Injury Assist System.

Key Features Implemented:

* User Registration and Login
* File Handling
* Injury Selection
* Treatment Recommendation
* Appointment Scheduling
* Report Generation
* Graphical User Interface (GUI)

Programming Concepts:

* Inheritance
* Encapsulation
* Abstraction
* Interfaces
* Composition
* Object-Oriented Design
