# Injury Assist System

## Overview

The Injury Assist System is a Java-based application designed to assist users in identifying sports-related injuries, determining their severity levels, receiving appropriate treatment recommendations, scheduling doctor appointments, and generating detailed injury reports.

The project was developed as an Object-Oriented Programming (OOP) project and demonstrates the practical application of software engineering concepts in the healthcare domain.

---

## Project Objectives

* Assist users in identifying injury types.
* Determine injury severity levels.
* Recommend suitable treatments based on the injury.
* Allow users to book medical appointments.
* Generate summarized injury reports.
* Store and validate user account information.

## Object-Oriented Programming Concepts Used

### Inheritance

The `User` and `Doctor` classes inherit from the `Person` class.

### Encapsulation

Attributes are declared private and accessed through methods.

### Abstraction

The `Report` abstract class defines shared report behavior.

### Interface

The `ReportInterface` interface is used to define report-related functionality.

### Composition

The `Report` class combines user, injury, treatment, appointment, and sport objects.

### Polymorphism

Abstract methods and object inheritance are used to support extensibility.
